TROUBLESHOOTING:
================================================================================

Python script fails with "ModuleNotFoundError: No module named 'requests'"
  SOLUTION: pip install requests

Bash script fails with "curl: command not found"
  SOLUTION: Use Python script instead, or install curl

Script fails with authentication error
  SOLUTION: 
    1. Verify GITHUB_TOKEN is set: echo $GITHUB_TOKEN
    2. Check token hasn't expired
    3. Verify token has 'repo' scope
    4. Generate new token if needed

Script succeeds but settings not visible in GitHub
  SOLUTION:
    1. Refresh the GitHub repository settings page
    2. Verify you have admin access
    3. Check the correct repository (Fused-Gaming/swords2silenced)
