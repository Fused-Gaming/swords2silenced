# Research Papers

This directory contains PDF files for research papers and supplemental analysis documents.

## Files

- `Veteran_Service_Paradox.pdf` - "The Veteran Services Paradox: Marketing Speech for Political Gains" by C. B. Tinker (April 2026)

## Adding PDFs

To add a research paper PDF:

1. Place the PDF file in this directory
2. Update the filename reference in `/src/pages/research-paper.tsx` if needed
3. Update the metadata (title, author, DOI, etc.) in the page component

## Notes

PDFs in this directory are served statically and can be:
- Viewed directly in the browser via the PDF viewer component
- Downloaded by users via the download button
- Referenced in documentation and citations

Ensure all PDFs are properly formatted and compress large files when possible to maintain optimal site performance.
